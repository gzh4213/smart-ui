## 特性

- Monorepo 风格 (pnpm workspace)
- 独立组件库
- 基于 Vue3 框架
- 支持 JSX 与 Vue 单文件组件
- Jest + Vue3 plugins 实现单元测试
- Eslint + Prettier + Husky 语法检查
- Vitepress + Vercel 文档网站搭建
- 基于 Action CI 实现持续集成与交付
